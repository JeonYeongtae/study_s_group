# 📝 데이터베이스 정리

## 📚 목차

### 6.1 데이터베이스의 큰 그림

- 📌 데이터베이스(Database)란?
- 📌 DBMS(Database Management System)
  - 🔸 대표적인 DBMS
- 📌 파일 시스템과 DBMS의 차이
- 📌 트랜잭션(Transaction)
  - 🔸 ACID: 트랜잭션의 4가지 성질

### 6.2 RDBMS의 기본

- 📌 RDBMS (Relational Database Management System)
- 📌 테이블(Table)
- 📌 기본 키(Primary Key)
- 📌 외래 키(Foreign Key)
- 📌 관계(Relationship)
- 📌 무결성 제약 조건 (Integrity Constraints)
- 📌 SQL 기초
  - 테이블 생성
  - 기본 조회
  - 삽입
  - 외래 키 제약
